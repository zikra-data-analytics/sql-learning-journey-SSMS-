# Company Employee Data Analysis (SQL Project)

## Project Overview
This SQL project analyzes employee data within a company database. It demonstrates the use of SQL aggregation and grouping functions to extract business insights.

## Database Table
- employeeDetails

## Analysis Performed
The project includes queries for:

- Employee count by department
- Employee count by gender
- Salary statistics (SUM, MIN, MAX, AVG)
- Department level payroll analysis
- Filtering using WHERE and HAVING clauses
- String aggregation of employee names

## SQL Concepts Demonstrated
- GROUP BY
- HAVING
- Aggregate functions
- ORDER BY
- Data filtering
- Temporary tables

## Tools Used
- Microsoft SQL Server
- SQL Server Management Studio (SSMS)
