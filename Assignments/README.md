# SQL Certification Training - Assignments 2 & 3

This repository contains my solutions for the SQL Certification Training assignments, focusing on Data Definition Language (DDL), Data Manipulation Language (DML), and advanced table Joins.

## 🛠️ Concepts Covered
- **Database Schema Design:** Creating structured tables with appropriate data types.
- **Data Filtering:** Using `WHERE` clauses, `LIKE` operators, and wildcards (`%`).
- **Data Manipulation:** Performing `UPDATE` operations on existing records.
- **Relational Joins:** Implementing `INNER`, `LEFT`, `RIGHT`, and `FULL OUTER` joins to combine data across tables.

## 📋 Assignment 2: Basic SQL Operations
**Objective:** Create a customer database and perform specific data retrieval tasks.

### Key Tasks:
1. Created a `customer` table including contact and location details.
2. Filtered customers based on naming patterns and specific cities (e.g., "San Jose").
3. Identified users based on email domains (Gmail) and specific character exclusions in last names.

## 📋 Assignment 3: Joins and Data Updates
**Objective:** Manage an `Orders` database and link it to the customer base.

### Key Tasks:
1. Designed and populated an `Orders` table.
2. Performed various Joins to identify:
   - Customers with active orders (**Inner Join**).
   - Customers without orders (**Left Join**).
   - Orders without registered customers (**Right Join**).
   - A complete overview of the business ecosystem (**Full Outer Join**).
3. Updated order amounts based on specific customer criteria.
